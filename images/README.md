# /images

Drop your project screenshots and assets here.

Recommended naming:
- og-image.jpg          — 1200×630 Open Graph card
- cs01-hero.jpg         — Streaming Platform hero image
- cs01-flow.jpg         — Flow diagram
- cs01-screen-*.jpg     — Individual screens

All images should use:
- loading="lazy"
- width / height attributes (prevents layout shift)
- meaningful alt text
